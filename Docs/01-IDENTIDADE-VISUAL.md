## ✨ Estética "Liquid Glass"
Qualquer elemento de interface deve seguir estas propriedades:
1. **Backdrop Blur:** `blur-[16px]` para todos os cards.
2. **Glass Shine:** Efeito de reflexo (shine animation) em botões primários no hover.
3. **Bordas:** `border border-white/10` para simular vidro.
4. **Gradientes:** Uso de gradientes radiais suaves no fundo da página para profundidade.
